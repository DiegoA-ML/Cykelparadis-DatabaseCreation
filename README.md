# Cykelparadis Database Project – DTU 02327

## Overview

This repository contains all materials for the **Cykelparadis group project** for the DTU course:

**Introductory Databases and Database Programming (02327, Fall 2025)**

The project demonstrates the design, implementation, population, and manipulation of a relational database for a bike repair shop. It includes:

- Conceptual and logical database design
- Table creation and population scripts
- SQL queries and data modifications
- SQL programming constructs (triggers, functions, procedures)
- Documentation of group members’ contributions

## Project Purpose & Objective

**Purpose:**  
To implement a fully functional relational database for a bike repair shop, demonstrating correct data modeling, population, query creation, and SQL programming capabilities.

**Objective:**  
- Design a normalized database schema capturing bikes, parts, customers, and repairs  
- Implement the schema in MariaDB with correct constraints  
- Populate the database with representative sample data  
- Demonstrate the use of SQL for table modifications, queries, and automation (triggers, procedures, functions)  
- Document the process and contributions in a final report
